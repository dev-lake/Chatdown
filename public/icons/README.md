# Icon Placeholder

This directory should contain the following icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can create these icons using any image editor. They should represent the Chatdown extension (e.g., a document or markdown symbol).

For quick testing, you can use any PNG images renamed to these filenames, but for production use, create proper icons at the specified dimensions.
